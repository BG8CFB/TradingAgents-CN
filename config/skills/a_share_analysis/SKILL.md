# A股深度分析技能

## 概述

本技能提供一套系统化的A股分析框架，指导分析师从多个维度对A股上市公司进行全面评估，并给出明确的投资建议。

## 分析框架

### 第一步：基本面核查

使用以下工具按顺序获取数据：

1. **`get_stock_fundamentals`** — 获取最新财务数据（PE、PB、ROE、净利润增速）
2. **`get_company_performance_unified`** — 获取最近 4 个季度的业绩数据（data_type="income"）
3. **`get_company_performance_unified`** — 获取业绩预告（data_type="forecast"）

核查要点：
- ROE > 15% 为优质，ROE < 5% 需警惕
- 净利润连续 3 年增长为正面信号
- 业绩预告增速与市场预期的偏差是重要催化剂

### 第二步：技术面分析

1. **`get_stock_data`** — 获取近 3 个月日线数据
2. 重点关注：
   - 均线系统（5/10/20/60日均线排列）
   - 量价关系（放量突破 vs 缩量下跌）
   - 支撑/阻力位

### 第三步：市场情绪与资金面

1. **`get_stock_sentiment`** — 获取情绪分析
2. **`get_money_flow`** — 获取资金流向（query_type="stock", ts_code=股票代码）
3. **`get_dragon_tiger_inst`** — 如有异动查看龙虎榜

### 第四步：宏观与板块背景

1. **`get_china_market_overview`** — 市场整体环境
2. **`get_macro_econ`** — 关键宏观指标（indicator="cpi" 或 "ppi"）

### 第五步：新闻与舆情

1. **`get_stock_news`** — 最新股票新闻（max_results=15）
2. **`get_finance_news`** — 行业相关财经新闻

## 输出格式

完成数据收集后，按以下结构输出分析报告：

```
## [股票代码] [公司名称] 深度分析报告

**分析日期**: YYYY-MM-DD
**市场环境**: [当前市场整体判断]

### 一、基本面评估
- 估值水平：PE/PB/行业对比
- 盈利质量：ROE趋势、现金流
- 成长性：营收/净利润增速

### 二、技术面分析
- 趋势判断：[上升/震荡/下降]
- 关键价位：支撑位 X.XX 元，阻力位 X.XX 元
- 量价特征：[描述]

### 三、资金与情绪
- 主力资金：[流入/流出]金额
- 市场情绪：[乐观/中性/悲观]

### 四、风险提示
- [列举2-3个主要风险]

### 五、投资建议
- 评级：[强烈推荐/推荐/中性/回避]
- 目标价：X.XX - X.XX 元（3-6个月）
- 建仓策略：[分批/一次性，建议仓位比例]
```

## 注意事项

- **数据时效性**：始终使用最新数据，不要引用超过 30 天的旧数据
- **A股特殊规则**：注意涨跌停（±10%，ST股±5%）、T+1交易制度
- **政策敏感性**：A股受政策影响大，重大政策出台需重新评估
- **流动性风险**：小市值股票（市值<50亿）需特别注意流动性风险
- **信息不对称**：国内新闻与实际经营可能有偏差，交叉验证财报数据
